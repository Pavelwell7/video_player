# Видеоплеер

Проект кастомного видеоплеера на HTML, CSS и JavaScript.

![video_player](https://github.com/user-attachments/assets/f2512855-3278-4502-af0d-f83b0f873305)

## Возможности и структура

- Гибкая вёрстка на Flexbox
- Основные стили лежатат в папке `css` в файле `styles.css`

### Кнопочки

Иконки лежат в папке `css` в  файле `font-awesome.min.css`, шрифты в папке `fonts`:

- Play / Pause
- Включение и выключение звука
- Полноэкранный режим

## Запуск проекта

- Сайт с прототипом видеоплеера можно посмотреть ссылке [video_player](https://pavelwell7.github.io/video_player/).
- Скопировать репозиторий и открыть файл `index.html` в любом браузере.
